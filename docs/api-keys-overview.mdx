API Keys is a module that allows you to create API Profiles with custom privileges such as *Read*, *Write* or *None* for all Modules in the Admin Portal. Once an API Profile is created you can generate an API Key which can be used in requests.

API Keys can also be disabled so that requests can no longer be made with a specific API Key or it can be reset which generates a new API Key for the API Profile.

If you are unsure of what API Keys are and what they are used for then you can visit the [Obtaining an API Key (Agilit-e Token)](/docs/obtaining-an-api-key-agilit-e-token) section as well as lookup what an [Application Programming Interface Key](https://en.wikipedia.org/wiki/Application_programming_interface_key) is and how it works.
