Data Mapping is a Module that can be used to convert data from one format to another, for example you can iterate / map over a JSON Object's properties and populate an Excel Spreadsheet accordingly, or you can populate / create a JSON Object based on data in the Excel Spreadsheet. You can also convert one form of a JSON Object into another.

This is a very useful way to produce excel spreadsheets in a programmatic way. This allows people to easily create Excel spreadsheets in Agilit-e rather than having to develop functionality to handle excel spreadsheets (which can be very tiresome and tedious)

## Source Types

* JSON
* Excel

## Destination Types

* JSON
* Excel
