## /encodeXML

* /encodeXML is used to encode XML Data. An [Agilit-e Admin Portal](https://portal.agilite.io/) Module configuration is not required.

## /decodeXML

* /decodeXML is used to decode XML Data. An [Agilit-e Admin Portal](https://portal.agilite.io/) Module configuration is not required.

## /XMLToJS

* /XMLToJS is used to convert XML to JSON. An [Agilit-e Admin Portal](https://portal.agilite.io/) Module configuration is not required.

## /JSToXML

* /JSToXML is used to convert JSON to XML. An [Agilit-e Admin Portal](https://portal.agilite.io/) Module configuration is not required.

## /html2json

* /html2json is used to convert HTML to JSON. An [Agilit-e Admin Portal](https://portal.agilite.io/) Module configuration is not required.

## /generatePDF

* /generatePDF API is used to generate a PDF Document based on the passed PDFMake JSON Schema. An [Agilit-e Admin Portal](https://portal.agilite.io/) Module configuration is not required.

## /generateUUID

* /generateUUID is used to generate a Unique number. An [Agilit-e Admin Portal](https://portal.agilite.io/) Module configuration is not required.

## /formatDateTime

* /formatDateTime is used to Format Date and Time in a specific format that is specified when calling the API. An [Agilit-e Admin Portal](https://portal.agilite.io/) Module configuration is not required.
