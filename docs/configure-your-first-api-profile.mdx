1. Create a new API Key Profile using the `Create New` button.

![704](/images/docs/8a92dde-API_Keys_Create_New.png "API_Keys_Create_New.png")

2. In the **General** section populate the `Profile Key` for the API Key Profile.

![1226](/images/docs/d6f6ee9-API_Keys_Profile_Key.png "API_Keys_Profile_Key.png")

3. Customise Privileges for the API Key Profile in the **Privileges** Section. There are more privileges listed than those shown below (due to image size constraints we cannot include them all)

![642](/images/docs/9fb4879-Better_Screenshot.PNG "Better_Screenshot.PNG")

4. Click on "Submit" to save your new Roles profile.
