---

[Suggest Edits](/edit/obtain-a-openweathermap-api-key)

1. Go to [OpenWeatherMap](https://openweathermap.org)
2. Sign In using your OpenWeatherMap credentials.
3. Navigate to the API Keys Tab.

![1642](/images/docs/13b059f-OpenWeatherMap-API-Tab.png "OpenWeatherMap-API-Tab.png")

4. Create a new key called "Admin Portal" and click on the `Generate` button.

![2008](/images/docs/09ac290-OpenWeatherMap-Generate.png "OpenWeatherMap-Generate.png")

5. Copy and Save this Key.

![1994](/images/docs/e95aca8-OpenWeatherMap-API-Key.png "OpenWeatherMap-API-Key.png")
