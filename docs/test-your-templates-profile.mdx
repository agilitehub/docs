Go to the Agilit-e API Reference page.

1. Navigate to the Templates Execution APIs category under the Agilit-e Templates section, followed by clicking on the `/execute` menu item.

![542](/images/docs/da8857a-Screen_Shot_2018-10-11_at_07.29.11.png "Screen Shot 2018-10-11 at 07.29.11.png")

2. Click on the user icon on the top right of the API logic and populate the generated Agilit-e token into the api-key field.

<Warning>
  **Note:** You will need to create an API Key, as mentioned earlier in the [API Keys Module](/docs/api-keys-overview) section of the documentation. Once the API Key is provided for one API, it will exist for the rest of the APIs.
</Warning>

![700](/images/docs/24f860e-api-key-authentication.png "api-key-authentication.png")

3. In the Headers section provide the Profile Key.

![2036](/images/docs/bcc9f21-Templates-Headers.png "Templates-Headers.png")

4. Click on the Try It button to execute the API.

* The Response Data will contain the Template Data (HTML)

![772](/images/docs/d9bc530-Templates-Response.png "Templates-Response.png")
