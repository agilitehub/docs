1. Create a new Templates Profile using the `Create New` button.

![702](/images/docs/20546a9-Roles-Create-New.png "Roles-Create-New.png")

2. Populate the Roles Profile

* Role Name: Manager
* Responsible User: John Doe
* Conditional Levels: IT

Overview of this Role Profile:

* John Doe is the Manager of the IT Department

You can create multiple profiles with the same Role Name but different Levels and Responsible Users, in some cases there might be more than one manager in an organisation just in different Levels / Departments.

![2810](/images/docs/fc799b7-Roles-Profile.png "Roles-Profile.png")

3. Click on "Submit" to save your new Roles profile.
