---
title: "JSON to Excel Data Mapping Profile"
description: "In this section you will learn how to create a Data Mapping profile that maps over a JSON Object and populates an Excel Spreadsheet"
---

[Suggest Edits](/edit/configure-your-first-data-mapping-profile)

1. Create a new Data Mapping profile using the Create New button.

![826](/images/docs/e2c8973-Data-Mapping-Create-New.png "Data-Mapping-Create-New.png")

2. Populate the Data Mapping profile: *General Tab*

* Profile Key: json2excel
* Profile Name: JSON To Excel
* Source Type: JSON
* Destination Type: Excel

![2798](/images/docs/d5c9c94-Data_Mapping_Form_General_JSON2Excel.png "Data_Mapping_Form_General_JSON2Excel.png")

3. In the Source Details Tab specify the *Starting Point* of the JSON object that was passed. In this case we are going to pass a JSON Object with a *data* property where we will start to map the data into the Excel Spreadsheet.

![1762](/images/docs/42b4681-Data_Mapping_Source_Details_JSON2Excel.png "Data_Mapping_Source_Details_JSON2Excel.png")

4. In the Destination Details Tab enter the *File Name* of the Excel File that will be used to populate.

![1746](/images/docs/bfa3e4a-Data_Mapping_Destination_JSON2Excel.png "Data_Mapping_Destination_JSON2Excel.png")

5. Navigate to the Field Mappings Tab and Create a new Field Mapping by clicking on the `Create New` button.

Field Mappings will contain the configuration of each field you want to iterate/map over in the passed JSON Object.

![2354](/images/docs/e57da14-Data_Mapping_New_Field_Mapping.png "Data_Mapping_New_Field_Mapping.png")

6. Populate the Field Mapping Form

General Details

* Mapping Name: *username*

Source Details

* Attribute Name: *name* (Property in JSON Object)

Destination Details

* Excel Worksheet Name: *worksheet1* (Name of the worksheet)
* Cell Name: *A1* (The Cell you want to populate in the Excel Spreadsheet)

![1980](/images/docs/e9dc133-Data_Mapping_Field_Mapping_JSON2Excel.png "Data_Mapping_Field_Mapping_JSON2Excel.png")

7. Click on the "Ok" button to save the Mapping Profile.

![2340](/images/docs/9dc0f86-Data_Mapping_Field_Mapping_View_JSON2Excel.png "Data_Mapping_Field_Mapping_View_JSON2Excel.png")

8. Click the "Submit" button to save and close the Data Mapping Profile.
