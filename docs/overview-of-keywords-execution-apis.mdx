## /getByProfuleKey

* The [/getByProfileKey](/reference#getbyprofilekey) API returns the Keyword profile where the `Profile Key` matches the `Profile Key` of the profile that was setup in the [Agilit-e Admin Portal](https://portal.agilite.io/).

## /getProfileKeysByGroup

* The [/getProfileKeysByGroup](/reference#getprofilekeysbygroup) API returns all Keyword profiles where the `Group Name` matches the `Group Name` that was setup in the [Agilit-e Admin Portal](https://portal.agilite.io/).

## /getLabelByValue

* The [/getLabelByValue](/reference#getlabelbyvalue-1) API returns a Keyword Label for a specific profile based on a `Value` key provided that was setup in the [Agilit-e Admin Portal](https://portal.agilite.io/) profile.

## /getValueByLabel

* The [/getValueByLabel](/reference#getvaluebylabel-1) API returns a Keyword Value for a specific profile based on a `Label` key provided that was setup in the [Agilit-e Admin Portal](https://portal.agilite.io/) profile.
