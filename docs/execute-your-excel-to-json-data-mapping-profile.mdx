1. Using Postman provide the following headers in your request

* api-key - Unique Agilit-e Generated API Key
* profile-key - excel2json (Profile Key of the Data Mapping Profile)
* Content-Type: application/json

<Warning>
  **Note:** You will need to create an API Key, as mentioned earlier in the [API Keys Module](/docs/api-keys-overview) section of the documentation. Once the API Key is provided for one API, it will exist for the rest of the APIs.
</Warning>

![2262](/images/docs/50829bb-excel2json_headers.png "excel2json_headers.png")

2. Click on the "Send" button to execute the Data Mapping Profile

![636](/images/docs/394a573-Send_button_postman.png "Send_button_postman.png")

## Response

The response will contain the following data. The Data Mapping Profile read from the Excel Spreadsheet and created a JSON Object based on what it read.

<CodeGroup>
  ```json json
  {
      "username": "John Doe"
  }
  ```
</CodeGroup>
