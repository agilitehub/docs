---

**Roles** is a Profiling Tool that allows the use of Roles to identify certain people. These roles can have multiple levels, for example:

* Manager Role for the level IT
* Manager Role for the level HR

These Roles have a Responsible Person property that defines the Responsible Person for a specific Role.
