1. Create a new Tier Structure profile using the `Create New` button in the "Tier Structure Profiles" card. This will open a new Tier Structure profile form.

![820](/images/docs/96168c0-Tier_Structures_Create_New.png "Tier_Structures_Create_New.png")

2. Populate the Tier Structure profile **Provide the following in the Form General Tab:**

* Profile Key: country **Label/Value Pair**
* Label: United States of America - Value: USA

The `Blue Square` represents the current tier you are in.

![3288](/images/docs/07da051-Tiers_Form.png "Tiers_Form.png")

3. Click on the Tier Entries Tab (Above Image).

4. Create a new Tier using the `Create New` button. This will open a new Tier Form.

![884](/images/docs/8139add-Tiers_entry_create_new.png "Tiers_entry_create_new.png")

5. Populate the Tier Profile **Provide the following in the Form General Tab:**

* Profile Key: states **Label/Value Pair**
* Label: Montana - Value: MT
* Label: Washington - Value: WA
* Label: - New York Value: NY

![3256](/images/docs/3e0e9b0-Tier2_Form.png "Tier2_Form.png")

Note: The `Blue Square` represents the previous tier and the yellow square represents the current tier.

6. Click the `Ok` button to Save and Close the current tier and go back to the previous tier.

![470](/images/docs/b17ae5b-Tier2_Ok_button.png "Tier2_Ok_button.png")

7. Click the `Submit` button to save and close the Tier Structure Profile.
