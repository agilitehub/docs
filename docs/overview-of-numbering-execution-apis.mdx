  Remember that the following Numbering Execution APIs are part of the [RESTful operations](https://en.wikipedia.org/wiki/Representational_state_transfer)(GET, PUT, POST, DELETE etc) that we mentioned in [Admin Portal vs API Server](/docs/admin-portal-vs-api-server)

## /generate

* The /generate generates a unique number based the on various conditions that you configured in the module via the [Agilit-e Admin Portal](https://portal.agilite.io/).

## /resetCounters

* The /resetCounters API resets all the numbering counters to their initial starting point as configured via the [Agilit-e Admin Portal](https://portal.agilite.io/).
