[Suggest Edits](/edit/generate-your-first-api-key)

1. In the API Keys list view click on the `Key` Icon in the *Actions* column to generate the API Key for the API Key Profile.

<Warning>
  The green `Key` Icon in the *Actions* column will only display on hover over. You will need to move your mouse over the area before the key is displayed.
</Warning>

![1442](/images/docs/1308337-Agilit-e_Portal_API_Keys_-_Generate_1.png "Agilit-e_Portal_API_Keys_-_Generate_1.png")

2. This will open up a dialog window in which you will find an *Icon that consists of two pages* also known as the `copy to clipboard` icon. Once you click this button, the API Key will be saved to your clipboard (ie you can paste it somewhere else)

![1329](/images/docs/97ab730-Agilit-e_Portal_API_Keys_-_Generate_2.png "Agilit-e_Portal_API_Keys_-_Generate_2.png")
