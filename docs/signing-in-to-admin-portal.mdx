## Overview

The Sign-In form displays by default on Agilit-e's Anonymous Landing Page (i.e. it displays by default if you have not yet logged in).

To sign in, enter your Agilit-e account's email address and password.

<Warning>
  **Note:** The Agilit-e Admin Portal has a forgot password feature, on click a password reset link will be sent to you via the email that you provided.
</Warning>
