## /generate

* The [/generate](/reference#generate-1) generates a unique number based the on various conditions that you configured in the module via the [Agilit-e Admin Portal](https://portal.agilite.io/).

## /resetCounters

* The [/resetCounters](/reference#resetcounters) API resets all the numbering counters to their initial starting point as configured via the [Agilit-e Admin Portal](https://portal.agilite.io/).
