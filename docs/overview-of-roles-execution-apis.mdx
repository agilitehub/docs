## /getRole

* The [/getAssignedRoles](/reference#getrole) API returns a list of responsible users as an array based on the `Role` and `conditional levels` provided in the [Agilit-e Admin Portal](https://portal.agilite.io/) module configuration.

## /changeConditionalLevels

* The [/changeConditionalLevels](/reference#changeconditionallevels) API changes the Conditional Levels of a Role Profile relevant to the `record-id` provided in the [Agilit-e Admin Portal](https://portal.agilite.io/) module configuration.

## /reAssignResponsibleUser

* The [/reAssignResponsibleUser](/reference#reassignresponsibleuser) API reassigns the Responsible User of a Role Profile relevant to the `record-id` provided in the [Agilit-e Admin Portal](https://portal.agilite.io/) module configuration.
