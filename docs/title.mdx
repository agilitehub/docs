---
title: "Configure Your First Number"
description: "In this section you will learn how to create a Numbering profile"
---

In the following example, we will profile a number for Leave Requests:

1. Create a new Numbering profile using the `Create New` button.

![762](/images/docs/26bb31a-Create-New-Numbering.png "Create-New-Numbering.png")

2. Populate the Numbering profile:

* Profile Key: leave\_numbering
* Profile Name: Leave Numbering
* Group Name (Optional): Could be "Internal Numbering", etc.

![1570](/images/docs/7b641c8-Profile-Key-Numbering.png "Profile-Key-Numbering.png")

3. Click on "Submit" to save the Numbering profile.
