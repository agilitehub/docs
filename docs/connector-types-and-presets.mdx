A Connector Type is the kind of Connector you want to create. By default, Agilit-e offers a "Web API" Connector Type, which essentially allows you to configure any kind of Web API you can think of.

Agilit-e also offers various Connector Types that are not Web APIs. Examples of this can be seen below. These Connector Types provide services that would not be regarded as Web APIs, but which Agilit-e converts into APIs.

## Agilit-e currently has the following Connector Presets:

* [Web API (Default)](https://en.wikipedia.org/wiki/Web_API)
* [FTP](https://en.wikipedia.org/wiki/File_Transfer_Protocol)
* [LDAP](https://en.wikipedia.org/wiki/Lightweight_Directory_Access_Protocol)
* [MongoDB (Mongoose)](https://mongoosejs.com/docs/api/model.html)
* [Microsoft SQL](https://en.wikipedia.org/wiki/Microsoft_SQL_Server)
* [PostgreSQL](https://en.wikipedia.org/wiki/PostgreSQL)
* [SMTP](https://en.wikipedia.org/wiki/Simple_Mail_Transfer_Protocol)
* [SSH2-SFTP](https://www.npmjs.com/package/ssh2-sftp)
