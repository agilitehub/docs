## Disable API Key

1. In your API Key Profile click on the `ellipse` icon on the *top right* of the API Key form to open the actions menu.

![1372](/images/docs/cdc640e-Disable_API_Key.png "Disable API Key.png")

2. Click on the `Disable API Key` Menu Item to disable the API Key Profile.

![196](/images/docs/22c5040-Disable_API_Key_2.png "Disable API Key 2.png")

*Note:* The API Key will now be disabled an you can no longer use it until it is enabled again.

## Reset API Key

1. In your API Key Profile click on the `ellipse` icon on the *top right* of the API Key form to open the actions menu.

![1372](/images/docs/bb6f4ce-Disable_API_Key.png "Disable API Key.png")

2. Click on the `Reset API Key` Menu Item to rest the API Key Profile.

![196](/images/docs/16c8b86-Reset_API_Key.png "Reset API Key.png")

*Note:* The API Key will now differ from the previous API Key when it is generated.
